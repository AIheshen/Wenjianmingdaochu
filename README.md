# 📘 文件名导出工具
## 作者：艾合
## 更新时间：2025-11-03
## 构建支持：ChatGPT · GPT-5
> 一款轻量、简洁、可视化的文件夹内容导出工具 —— 支持层级结构展示、自动输出到下载目录、含图标界面。

## 🧩 功能介绍

| 功能 | 说明 |
|------|------|
| 📁 选择任意文件夹 | 选择你想导出文件名的目录 |
| 💾 自动导出到下载目录 | 默认导出到系统的 **下载文件夹** |
| ✏️ 智能命名 | 自动生成文件名：`文件夹名（内所有文件名称）.txt` |
| 🌳 层级结构展示 | 文件夹前有 📂 emoji，显示清晰的层级结构 |
| 📜 可选完整路径 | 勾选“包含完整路径”时，每行显示完整路径（不缩进） |
| 🧭 可自定义导出目录 | 仍可手动修改导出 TXT 的保存位置 |
| 🪟 可视化界面 | 使用 tkinter + Pillow，图标与布局简洁美观 |
| 🔔 自动打开导出目录 | 导出完成后自动打开保存位置 |
